# Contributing to Flame Spear Engine
Flame Spear Engine is an open source graphics engine that is written in C++. 
The goal of this engine is to spark creativity in the game design industry as well as the engine industry.
This engine is written as much as it can in pure standard C++ although it uses Vulkan to interact with the GPU. 


## Ways to contribute
Given that Flame Spear Engine is an open source graphics engine the community would greatly appreciate contributors in the community.
1. Reporting bugs / Suggesting features
2. Creating PRs to solve a problem you have
3. Asking to work on a reported issue

## Reporting bugs / Suggesting features
### How to report bugs
This section shows how to go about reporting a bug: 
1. Go to this link : https://github.com/MurderBot1/Flame-Spear-Engine/issues/new/choose
2. Click bug report
3. Follow instructions on that page

### How to suggest features
This section shows how to go about suggesting new features:
1. Go to this link : https://github.com/MurderBot1/Flame-Spear-Engine/issues/new/choose
2. Click feature request
3. Follow instructions on that page

## Creating a PR (Pull Request)
To create a PR we would appreciate it if you followed these steps
1. Go to this link: https://github.com/MurderBot1/Flame-Spear-Engine/pulls
2. Describe exactly what you changed and why you did in 1-3 paragraphs minimum
3. Add an image of the changes
4. Make sure all of the tests pass
5. Once everything is in order message murderbot1@outlook.com with the PR link to be reviewed

## Fixing an issue
To request to fix an issue on Flame Spear Engine follow these steps
1. Go to the issues tab on GitHub
2. Find an unclaimed issue that you would like to work on
3. Ping @MurderBot1 in the comments with the amount of time you thing it will take
4. After this wait until we reply back and assign you (and decide who is the best fit)
5. You will have that much time plus 48 hours to get a working version
